## Workshops and training

### R Workshop using publicly available data (Hampton University, 2024)

- **Workshop I (Day 1, morning)**: Data sourcing and wrangling
- **Workshop II (Day 1, afternoon)**: Exploratory data analysis
- **Workshop III (Day 2, morning)**: Interactive team-science applications of Day 1 wrangling + EDA techniques
- **Workshop IV (Day 2, afternoon)**: Advanced topics in statistical analysis and programming (informed by participant interests)
