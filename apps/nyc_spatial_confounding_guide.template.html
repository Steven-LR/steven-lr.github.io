Methods documentation

Data source: ACS 5-year tract-level estimates for NYC are retrieved in R via tidycensus using the Census API key, then joined to tract geometries from tigris by GEOID.

Variables: uninsured S2701_C04_001, poverty S1701_C03_001, unemployment S2301_C04_001. Invalid and suppressed values are filtered out during preprocessing.

Modeling: Act 2 uses naive OLS (uninsured ~ predictor) and maps residuals. Act 3 uses a BYM2-inspired spatial adjustment proxy that smooths residual structure over tract-neighbor relationships to separate predictor effect from place-based signal.

Rendering: this page is fully precomputed and self-contained. Runtime does not call Census or TIGER APIs; all geometry and model outputs are embedded in this HTML at build time.

ACT 2
A naive OLS model may look strong, but tract observations are not independent in space. The residual map checks whether unexplained signal still clusters geographically. If it does, the naive model is biased by spatial confounding.
